💳 Paga Eu - Prova de Conceito (PoC)
O Paga Eu é uma ferramenta de pagamento rápido e simplificado, projetada para ser uma solução de resiliência financeira em cenários onde a tecnologia convencional falha.

⛈️ Resiliência e Autonomia em Emergências
Esta ferramenta foi pensada estrategicamente para localidades com difícil acesso à internet ou situações de calamidade pública (como chuvas intensas, tempestades e desastres naturais). Quando a infraestrutura de rede é comprometida, a economia local não pode parar.

Operação Offline: Uma vez carregado no navegador, o sistema gera cobranças Pix sem depender de sinal de dados ou Wi-Fi.

Portabilidade Física: Por ser composto por um simples arquivo HTML único, o Paga Eu pode ser transportado via pen drive, cartões de memória ou outros meios físicos.

Kit de Emergência Digital: Pode ser distribuído facilmente em comunidades isoladas, garantindo que pagamentos e recebimentos continuem sendo processados mesmo sem conexão estável.

🚀 Funcionalidades Atuais
Geração de Pix Estático: Criação instantânea de payloads no padrão EMV QRCPS (Banco Central).

Interface Bancária: Design limpo, profissional e intuitivo, inspirado nos principais apps de finanças.

Máscaras Dinâmicas: Formatação automática em tempo real para CPF, Celular e Moeda (BRL).

Privacidade Total: Sem backend. Toda a lógica de geração de QR Code e cálculo de CRC16 acontece no navegador do usuário.

Aba de Apoio: Sistema de contribuição voluntária integrado.

🌍 Em Breve: Integração BRICS Pay
Estamos trabalhando para expandir as fronteiras do Paga Eu. Em breve, lançaremos uma versão com suporte ao BRICS Pay, permitindo cobranças internacionais e suporte nativo para moedas como Yuan (CNY), Rublo (RUB) e outras moedas do bloco.

📥 Como Utilizar ou Baixar
1. Clonando o Repositório
Bash
git clone https://github.com/seu-usuario/paga-eu.git
2. Uso Direto (Otimizado para Pen Drives)
Basta baixar o arquivo index.html. Ele é totalmente autossuficiente e não requer instalação de dependências ou servidores para funcionar.

📱 Transforme em um Aplicativo (PWA)
O Paga Eu funciona como um aplicativo nativo no seu celular sem precisar de lojas de apps.

Android (Chrome): Acesse o arquivo, toque nos três pontos e selecione "Instalar aplicativo".

iPhone (Safari): Toque em Compartilhar e selecione "Adicionar à Tela de Início".
